
# Multi-Agent AI Chatbot using LangGraph

An intelligent multi-agent chatbot built with **LangGraph** and **LangChain** that routes user queries to the appropriate AI agent, including LLM, RAG, Web Search, Vision, Image Search, and AI Image Generation.

---

# 🚀 Live Demo

https://multi-agent-chatbot-langgraph-k2lbzdyqymqmnr7bltxfae.streamlit.app/

---

#  Features

- 🤖 General AI Chat (LLM)
- 📚 RAG (PDF Question Answering)
- 🌐 Web Search
- 🖼️ Image Search
- 👁️ Image Understanding (Vision)
- 🎨 AI Image Generation
- 🧠 Intelligent Query Routing with LangGraph
- 💬 Chat History
- ⚡ Fast responses using Groq LLM

---

# 🛠 Tech Stack

- Python
- LangChain
- LangGraph
- Groq
- FAISS
- Hugging Face Embeddings
- Streamlit
- Pillow
- SerpAPI
- Pollinations AI (Image Generation)
